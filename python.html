Customizable Conway's Game of Life

I've always been interested in tweaking the rules behind Conway's Game of Life. I created a Jupiter notebook where you can fully customize the rules behind the game and create a custom initial set up. The different generations are saved as an animated .gif. The code can be found on my github page. The orginial rules are as follows

  • Birth rule: An empty cell with exactly three live neighbors becomes alive
  • Death rule: A live cell with zero or one neighbors dies of isolation, or with four or more neighbors dies of overcrowding
  • Survival rule: A live cell with two or three neighbors survives to the next generation
